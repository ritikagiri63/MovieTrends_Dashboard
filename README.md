ğŸ“– Story Line of the Project
The movie industry generates massive amounts of data related to budgets, revenues, genres, and audience ratings.  
However, raw data alone does not provide insights.  

This project transforms raw movie datasets into an interactive dashboard that:
- Identifies **high-revenue genres**
- Compares **production company performance**
- Evaluates **return on investment (ROI)**
- Highlights **profitable movie categories**

The goal is to uncover hidden trends and patterns that influence box office success.
